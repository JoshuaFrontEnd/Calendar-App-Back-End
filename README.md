- Node
- Express
- Mongoose
- Modelos
- Controladores
- Middlewares de Express
- JWT
- Autenticación pasiva
- Payload
- Encriptación
- Rutas
- CORS
- Revalidar tokens
- MongoDB
- Mongo Atlas
- Mongo Compass
- CRUD
- Validaciones automáticas y personalizadas
